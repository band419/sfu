# SFU - 三次样条插值自适应算法

这是一个实现三次样条插值的Python项目，支持自适应插值点选择。

## 功能

- **自适应插值点生成**：根据函数一阶导数的绝对值自动选择插值点
- **三次样条插值**：使用SciPy的CubicSpline实现高精度插值
- **理论误差计算**：计算插值的理论相对误差最大值

## 安装依赖

```bash
pip install numpy scipy
```

## 使用方法

运行主程序：

```bash
python main.py
```

## 项目结构

- `main.py` - 主程序，包含自适应插值和误差计算功能
- `sfu.drawio` - 项目相关的流程图或架构图

## 算法说明

该项目实现了基于一阶导数分布的自适应插值点选择算法，可以在函数变化剧烈的区域自动增加插值点密度，从而提高插值精度。
